# Start

## Auf dem Raspberry Pi

```bash
sudo python3 taupunkt.py
```

Web-UI im Browser:

```
http://<pi-ip>:8080/
```

Beenden: `STRG+C` (Overlay wird sauber entfernt, Lüfter geht aus).

## Auf dem Windows-PC (Vorschau ohne Hardware)

```powershell
py web_preview.py
```

Browser:

```
http://localhost:8080/
```
